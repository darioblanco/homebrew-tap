# homebrew-tap

Homebrew formulae for [darioblanco](https://github.com/darioblanco) projects.

## Install

```bash
brew install darioblanco/tap/pulpo
```

## Available formulae

| Formula     | Description                                   |
| ----------- | --------------------------------------------- |
| `pulpo`     | Pulpo daemon + CLI (recommended one-command) |
| `pulpod`    | Pulpo daemon only                             |
| `pulpo-cli` | Pulpo CLI only                                |
